Basic Commands

When start the terminal application you are given access to a shell. The default shell for most Linux distributions is `bash`.

The shell accepts commands from the user and makes system calls for execution. Many commands are installed when the system is intalled. If something additional is needed you can install it later.

<PATH/>COMMAND <SWITCHES> <ARGUMENTS>

Linux filesystems are case-sensitive by default.

Basic Commands. . .

If want to know how the commands work.

man rm

# q (quits), SPACE (forward page), b (back page)

info <COMMAND>    # get the info page for a command

info ip

vi

Vi has two modes insertion mode and command mode. When you enter the editor you are in command mode. You use insertion mode to make changes. After you are done with insertion mode you use [ESC] to return to command mode.

Find a cheat sheet that you like, print it out, and keep it next to your computer until you use the commands enough to remember them.

Start by typing 'vi FILENAME'

vi - basic commands

  • Quitting and Saving
    • :q -- exit if no changes have been made
    • :wq or :x -- exit saving(writing) changes
    • :q! -- exit not saving changes
  • Moving Around
    • up arrow or 'k' -- move up
    • down arrow or 'j' -- move down
    • left arrow or 'h' -- move left
    • right arrow or 'l' -- move right

vi - basic commands. . .

  • Inserting Text
    • i -- insert before cursor
    • a -- append after cursor
    • o -- append in a new line after the current line
    • r -- replace a character
    • cw -- change word
  • Deleting Text
    • x -- delete a character to the right of the cursor
    • dd -- delete the current line

vi - basic commands. . .

  • Yanking and Putting Text
    • yy -- yank the current line (copy)
    • dd -- delete the current line (cut)
    • p -- append after the position or after the line
    • P -- insert before the position or before the line
  • Searching
    • /STRING -- search for the first instance after cursor of a STRING (wraps)
    • n -- search again for the next instance
    • N -- search again for the previous instance
