HTTPd Overview

  • The Web, a.k.a WWW(World Wide Web)
  • Serves content over HTTP(Hypertext Transfer Protocol)
  • Content could be HTML, Javascript, Images, or other content.

  • HTTPd = HTTP Daemon
  • Apache httpd is a very common HTTPd Server
  • Apache Software Foundation

HTML Asside. . .

  • HTML = Hyper Text Markup Language

  • www.w3.org
  • www.w3schools.com

  • HTML is useful to know as an administrator

HTTPd Installation

  • Install
    • 
sudo apt-get update
sudo apt-get install apache2 -y
  • Test
    • 
# From a browser hit:
http://localhost
  • Remove
    • 
sudo apt-get remove apache2 -y
sudo apt-get autoremove -y

HTTPd Configuration

  • Main Configuration File
    • 
/etc/apache2/apache2.conf
  • ServerRoot
    • Specifies the top of the directory tree where all configuration files are stored. default /etc/apache2
  • DocumentRoot
    • Specifies the root directory of the served content. default: /var/www
  • ServerAdmin
    • Specifies the e-mail address that the server sends mail to when a user clicks the link on an error page.

HTTPd Configuration cont.

  • ServerName
    • Specifies the server name.
  • Lisen
    • Specifies the port that the server listens on. default: 80

HTTPd Administration

  • enable/start
    • 
sudo update-rc.d apache2 enable
sudo service apache2 start
  • restart
    • 
sudo service apache2 restart
  • stop/disable
    • 
sudo service apache2 stop
sudo update-rc.d apache2 disable

HTTPd Administration cont.

  • Modules
    • Modules add functionality.
    • Enable Modules
      • 
sudo a2enmod
    • Disable Modules
      • 
sudo a2dismod
