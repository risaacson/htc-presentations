for Loops

Used when you have a list or sequence of items that you want to iterate across.

Basic Loop in a Script


#!/bin/bash
for x in 1 2 3 4 5; do
  echo ${x}
done

What does this script do?


#!/bin/bash
for x in 1 2 3 4 5; do
  echo $(( ${x} + 1 ))
done

A script looping over a list of strings:


#!/bin/bash
for cheese_type in mozarella chedder gouda; do
  echo "I like eating ${cheese_type} cheese."
done

Adding new users with a script:


#!/bin/bash
for user_name in larry mike susan; do
  useradd ${user_name}
  echo "User ${user_name} added"
done

Scripts generating sequences:

Form 1: 


#!/bin/bash
# .. must be exactly as below
for x in {1..10}; do
  echo "${x}"
done

Form 2: 


#!/bin/bash
for x in $( seq 1 10 ); do
  echo "${x}"
done

Nested loops in a script:


#!/bin/bash
for x in $( seq 1 10 ); do
  for y in $( seq 1 10 ); do
    echo "${x} + ${y} = $((${x}+${y}))"
  done
done

while Loops

Used when you have a boolean operation that you want to execute commands until a desired state.

While loops continue until a condition is met:


#!/bin/bash
sum=0
end_at=10
while [ ${sum} -lt ${end_at} ]; do
  echo ${sum}
  sum=$(( ${sum} + 1 ))
done

While loops can be used to read lines from a file:


#!/bin/bash
while read line; do
  echo ${line}
done <file.txt
