Unix

  • 1969 - Ken Thompson, Dennis Ritchie and others started working on what was to become UNIX at Bell Labs
  • Intended to be a multi user operating system that originally ran on the Dec PDP-7
  • Originally written in Assembly Language and ported to C in 1973 which made it portable to other hardware
  • Originally had a CLI interface but in 1984 at MIT X was originated providing a GUI interface

Unix (cont.)

Hamstrung by AT&T and future intellectual property owners imposing strict licensing agreements. Early on there was a lot of sharing until the lawyers got involved.

Popular Unixes:

  • BSD
  • Sun Solaris
  • HP/UX
  • IBM AIX

Internet

  • 1961 - First paper on packet switching
  • 1969 - First nodes of ARPANET
  • 1971 - Ray Tomlinson of BBN invents an email program
  • 1972 - First computer-to-computer chat takes place at UCLA
  • 1979 - USENET established using UUCP between Duke and UNC

licensing

  • Under a consent decree in settlement of an antitrust case, Bell System (parent of Bell Labs) was forbidden from entering the computer business, so Ken Thompson started sending out tapes when requested
  • In 1983 the Justice Department broke up Ma Bell after settling a second antitrust case with AT&T which freed up Unix for commercialization
  • Unix became tightly controlled and hard to hack. This just about killed Unix and its children.

licensing (cont.)

  • Also in 1983 The GNU Projects was started by Richard Stallman(rms)
  • 1988-1989 the GNU Public License v1 is created

GNU = GNU is Not Unix

Four Software Freedoms:

  • (freedom 0) The freedom to run the program as you wish, for any purpose.
  • (freedom 1) The freedom to study how the program works, and change it so it does your computing as you wish. Access to the source code is a precondition for this.
  • (freedom 2) The freedom to redistribute copies so you can help your neighbor.
  • (freedom 3) The freedom to distribute copies of your modified versions to others. By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.
