Navigation with Bash


mkdir foo # Create Directory
cd; cd foo; cd ..; cd ~; cd /tmp # Navigate Directories
mkdir bar; rmdir bar # Make and Remove Directories
touch foobar; rm foobar # Make and Remove Files
mkdir barfoo; touch barfoo/bazfoo; rm -r barfoo
cp SRC DST; mv SRC DST # Copy and Move from a source
                       # file/directory to a destination
ps -fu USER; ps auxww; top # Look at processes

Virtualization

Carving up larger compute and storage into smaller logical systems. This way we can have more servers to build out systems on. Software is logically seporated from the host operating system.

  • Creation of Virtual Machines (VMs)
  • Emulator vs. Hypervisor
  • Full Virt vs. Para Virt vs. OS Level


resources:

  • Virtualization (Wikipedia)

Full Virtualization (Software/Hardware Assisted)

  • Simulation of systems hardware so an operating system and software run unmodified
  • May be assisted by hardware
    • Parallels site
    • VirtualBox site
    • VMware site

Paravirtualization

  • Some but not all of the systems hardware is emulated
  • As a result software may need to be modified to run
    • Xen site

OS Level Virtualization

  • Not really virtualization as no simulation is taking place
  • Process and resource isolation managed by software working with the kernel
    • FreeBSD Jails site
    • LXC/Docker site/site
    • Solaris Containers/Zones

Introduction to Vagrant

  • Allows us to start automating virtual machines
  • Configured through a Vagrantfile
    •

  config.vm.box = "centos/7"
  config.vm.hostname = "centos7.box"
end
  • Easy commands to remember for management
    • 
vagrant up
vagrant ssh
vagrant destroy

What does Vagrant get us?

  • Allows us to create the same VM consistantly
  • Allows us to incrementally develop systems
  • Super quick to create and destroy.

Vagrant Installation


sudo -i
cd /etc/yum/yum.repos.d/; wget http://download.virtualbox.org/virtualbox/rpm/el/virtualbox.repo
yum update -y
yum install -y binutils qt gcc make patch libgomp glibc-headers glibc-devel kernel-headers kernel-devel dkms
yum install -y VirtualBox-5.1
cd
wget 'http://download.virtualbox.org/virtualbox/5.1.4/Oracle_VM_VirtualBox_Extension_Pack-5.1.4-110228.vbox-extpack'
VBoxManage extpack install Oracle_VM_VirtualBox_Extension_Pack-5.1.4-110228.vbox-extpack --replace
wget 'https://releases.hashicorp.com/vagrant/1.8.5/vagrant_1.8.5_x86_64.rpm'
yum localinstall vagrant_1.8.5_x86_64.rpm
