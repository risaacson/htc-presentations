Review Basic Apache HTTPd Install

  • Installation
  • Enable/Disable starting at boot
  • Start/Stop the daemon

Installation

  • CentOS package is httpd
  • Ubuntu package is apache2
  • Must install as root

  yum install http -y

Enable/Disable starting at boot


## Enable
chkconfig httpd on
# or
systemctl enable httpd.service

## Disable
chkconfig httpd on
# or
systemctl disable httpd.service

Start/Stop the daemon


## start
service httpd start
# or
systemctl start httpd.service

## Disable
service httpd stop
# or
systemctl stop httpd.service

Check HTTPd is running


ps auxww | grep httpd

What is HTTP?

  • Hypertext Transport Protocol
  • A standard way of communicating hypertext.
  • Initiated by Tim Berners-Lee at CERN in 1989
  • Defined in many RFCs
  • Client/Server

  • Hypertext Transfer Protocol (Wikipedia)

The protocol that the World Wide Web is built on

HTTP Main Components

  • HTTP Session
  • HTTP Authentication/Security
  • HTTP Request
  • HTTP Response

HTTP Session

  • HTTP is built on a TCP connection
  • Insecure ports 80 or 8080
  • Secure ports 443

HTTP Authentication/Security

  • Authentication
    • username:password
    • Token
  • Security:
    • HTTPS / SSL

    • Restrict access with a firewall if you don't want people getting in

HTTP Request

  • Headers
  • Method
    • DELETE * GET * HEAD
    OPTION * POST * PUT
    Others. . .
  • Path
    • /html/2016-ccis2122-40/index.html
  • Payload
    • A file that you are uploading?

HTTP Response

  • Response Headers
  • Response Code
    • 200
    • 301
    • 404
    • 500
  • Response Body

What is HTTPd?

HyperText Transfer Protocol Daemon

  • i.e. a web server

Example of web servers

  • Apache HTTPd
  • Lighttpd
  • Nginx

There are others but for now we are focusing on Apache's HTTPd.

Apache HTTPd

  • Apache HTTP Server
  • Sometimes just called Apache

  • Apache HTTPd
  • Apache Webserver (Wikipedia)

History

  • Based off of NCSA HTTPd in early 1995(forked)
  • Developed and supported by a community of developers
  • is a part of the Apache Software Foundation
  • is Free and Open-Source Software

  • Estimated to serve traffic of half of all active websites

  • Stable is 2.4.33

Important Directories/Files

  • /etc/httpd - Configuration Files
  • /etc/httpd/conf/httpd.conf

  • /var/www - Document Root

  • /var/log/httpd/ - Log Files

cURL

  • Allows you to work with http at a low level

sudo yum install curl

curl http://www.google.com/
  • cURL
  • cURL (Wikipedia)

cURL is a Swiss-Army tool

  • Low-level access to HTTP
  • Standard GET queries are easy
  • Forms are Easy

cURL Examples.


curl https://presentations.recycledthoughts.org/html/index.html

curl https://presentations.recycledthoughts.org/html/index.html > /tmp/index.html

# will not follow links (redirects)
curl https://presentations.recycledthoughts.org/
# will follow links (redirects)
curl -L https://presentations.recycledthoughts.org/

cURL Examples. .


# Pass in Authentication
curl -u username:password http://example.com

# Verbose output!
curl -v https://presentations.recycledthoughts.org/

# Berbose but don't show progress meters or statistics
curl -sv https://presentations.recycledthoughts.org/

# RTFM
man curl
